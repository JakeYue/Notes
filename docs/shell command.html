1、EOF

EOF 把EOF内的内容作为输入给其他子命令或者子shell,知道遇到EOF结束

cat << EOF

/ --------------

这是个输入。     #

/ --------------

EOF

2、case

case 多分支条件判断语句,只能判断一种条件

#格式
case $变量名 in 
var 1)
    执行
    ;;
var 1)
    执行
    ;;
* )
    执行***
    ;;
esac

读取输入
